Mengenal Semantic Elements di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan bloc artikel-artikel yang ada di Website kita, tag article bisa digunakan element header dan footer. Karena tag article memiliki type blok, artinya element ini memiliki lebar sesuai lebar browser jika kita menentukan lebarnya

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita menambahkan aside di bagian bahwah postingan. Karena aside memiliki tipe blok maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
